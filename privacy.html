Privacy Policy

Last updated: May 3, 2026

Overview

Ver Fikrini operates an internal analytics tool ("VF Ads Analytics") that connects to TikTok Ads Manager via the official TikTok Marketing API. This policy describes how that tool handles data.

Scope

This tool is for internal use by Ver Fikrini's own team only. It is not a consumer-facing product. It is not distributed, sold, or licensed to any third party.

Data we access

Through the TikTok Marketing API, the tool retrieves performance metrics (spend, impressions, follows, clicks, video view rates, etc.) for advertising campaigns running on Ver Fikrini's own TikTok ad accounts. We access only aggregated, account-level performance data — no individual user information, no audience-level personal data.

Data we store

Retrieved metrics are stored locally on the developer's machine in a SQLite database. No data is uploaded to external servers, no data is shared with third parties, no data is sold.

Data we do not collect

We do not collect any data from end users, viewers, or members of the public. We do not run consumer-facing services. We do not use cookies or tracking technologies on this website beyond what GitHub Pages provides by default.

Third parties

The tool relies on the TikTok Marketing API. Use of TikTok services is governed by TikTok's own privacy policy.

Retention

Performance data is retained locally for as long as it is operationally useful. Authentication tokens are stored only as long as needed and revoked when no longer in use.

Contact

For questions about this policy, contact: info@verfikrini.com
